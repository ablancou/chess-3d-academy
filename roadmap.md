# Chess 3D Academy - Roadmap

Este documento centraliza las futuras mejoras e ideas pendientes para llevar la plataforma al siguiente nivel ("El mejor juego de Ajedrez 3D que haya existido").

## 1. Coach de Ajedrez en Tiempo Real (Estilo chess.com)
- **Evaluación de Jugadas:** Dar retroalimentación interactiva después de cada jugada (ej. "Jugada Excelente", "Imprecisión", "Error Grave").
- **Explicación Interactiva:** Explicar por qué una jugada fue buena o mala basándose en el análisis de Stockfish (WASM).
- **Sugerencias y Pistas:** Permitir al usuario solicitar una pista si no sabe qué mover.

## 2. Reconocimiento de Aperturas
- **Detección Automática:** Detectar la apertura actual utilizando la API de Lichess o un libro de aperturas local.
- **Información y Teoría:** Mostrar el nombre de la apertura y una breve descripción de sus objetivos.
- **Alternativas de Respuesta:** Mostrar visualmente cuáles son las mejores jugadas teóricas en la posición actual de la apertura.

## 3. Plan de Aprendizaje Progresivo (Estilo Duolingo)
- **Ruta de Progresión (Path):** Una vista de mapa con diferentes "mundos" o "niveles" (ej. Movimiento de piezas, Tácticas básicas, Finales, Aperturas).
- **Lecciones Interactivas:** Pequeños puzzles o escenarios específicos que enseñen conceptos gradualmente.
- **Sistema de Vidas y Rachas:** Mantener al usuario motivado a practicar todos los días mediante recompensas visuales (gemas, monedas o XP).
- **Gamificación Avanzada:** Usar el sistema actual de XP para desbloquear tableros y piezas, expandiendo los niveles (ej. misiones diarias o logros).
