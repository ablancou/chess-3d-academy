# ♟️ chess-3d-academy

**The ultimate 3D Chess learning platform** — built with Next.js 16, React Three Fiber and chess.js.

An immersive educational experience that lets players learn, practice, and analyze chess through high-fidelity 3D interactions.

## 🚀 Live Demo

[https://chess-3d-academy.vercel.app](https://chess-3d-academy.vercel.app) *(coming soon)*

## ✨ Current Features

- Fully interactive 3D chessboard
- Intuitive piece manipulation (mouse & touch)
- chess.js rules engine
- High-quality 3D rendering powered by React Three Fiber + Three.js
- Modern, responsive interface

## 🛠️ Tech Stack

- **Next.js 16** (App Router + Server Components)
- **React Three Fiber** + Three.js for 3D rendering
- **chess.js** for game logic
- TypeScript
- Tailwind CSS + modern component library
- Future integrations: Stockfish.wasm, real-time multiplayer, AI coaching
